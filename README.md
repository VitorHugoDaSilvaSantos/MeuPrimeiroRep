# MeuPrimeiroRep
## Vitor
## 1° passo
  Dentro do gitHub clique no Botão " code "
  ## 2° passo
  Vai aparecer um link =, copie e coloque no navegador
  ## 3° passo 
   Entre no explorar de arquivos e, vai em este computador e aperte em " u " , busque a pasta com seu nome 
 ## 4° passo
  Crie uma pasta chamada Git
  ## 5° passo
   Entre nela e aperte com o botão direito e va em " Abrir terminal " , se nao deu certo vai la em cima e escreva " cmd "
 ## 6° passo
  Dentro do terminal digite o codigo " git clone " de um espaço e insira o link que você copiou (https://github.com/VitorHugoDaSilvaSantos/MeuPrimeiroRep.git)
 ## 7° passo
  Depois disso você vai estar na pasta, use o codigo " cd " para poder entar no " MeuPrimeiroRep " " cd MeuPrimeiroRep "
 ## 8 ° passo 
  Apos isso coloque o codigo " code . " e entramos no VS code, no diretorio que estavamos, que era o MeuPrimeiroRep direto no README que marcamos na hora que estavamos criando o repositorio no GitHub
 ## 9° passo
  Escrevemos " ## nome (Vitor) " e alteramos o repositorio, por conta disso termos que salvar 
 ## 10° passo
  Teremos que voltar no cmd e digitar " git add . ", ele cria uma fila das alteração que fizemos para enviar com o commit para o GitHub
 ## 11° passo 
 Depois use o codigo " git commit -m " ( é um comentario do que fizemos, que no caso foi acrescentar " Adicione seu nome ")
 ## 12° passo
 Logo apos isso escreva o codigo " git push -u origin main " para poder enviar as alterção que fixemos direto no main do projeto
